Welcome to Carrot Farm

In this game you will learn to code. The programming language you will learn is Cascading Style Sheets (CSS). This language is used to design websites. By learning CSS you will also learn some basic skills needed to be a programmer.

In this game, you will be tending a carrot garden, and by writing code in CSS you will be able to water the carrots so they will grow.

Note to teacher or parent: this game is for children at the age of 10 and up. If the child is stuck on one of the levels try explaining to them what they need to do without telling them the answer.
